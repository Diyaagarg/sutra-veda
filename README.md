Sutra Veda – Premium Rakhi Collection Website

Sutra Veda is a modern, responsive e-commerce showcase website for premium handcrafted Rakhis and Friendship Bands. The website is designed to provide customers with a seamless browsing experience and direct ordering through WhatsApp.

✨ Features

- Responsive design for desktop, tablet, and mobile devices
- Modern and elegant UI inspired by Indian tradition
- Product category filtering
- WhatsApp-based ordering system
- Click-to-call functionality
- Product image zoom modal
- Smooth scroll animations
- Mobile-friendly hamburger navigation menu
- Customer reviews section
- Fast-loading static website

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Fonts

🚀 Deployment

This project is deployed using Vercel.

Run Locally

1. Clone the repository

git clone <repository-url>

2. Open the project folder

cd sutra-veda

3. Open "index.html" in your browser.

No build process or additional dependencies are required.

📱 Ordering

Customers can place orders directly through WhatsApp using the integrated order buttons available throughout the website.
